# food-tracker ДЗ 2
NutriHack — дневник питания, позволящий отслеживать съеденные калории и БЖУ. Этот проект — мобильная вёрстка главного экрана.
Экран содержит дату, статус выполнения нормы по калориям, блоки приёмов пищи и кнопку добавления.

Макет Figma: https://www.figma.com/design/S9TPe5ATrjnax3sP7Wwxfx/UCD?node-id=114-662&m=dev

Деплой: https://elisabelousova.github.io/food-tracker/

Использованные технологии:

**HTML + CSS**
- Семантическая HTML-вёрстка   
- CSS-переменные  
- Flexbox и абсолютное позиционирование  
- Адаптация под мобильный экран  

**JavaScript**
- Динамическое создание DOM-элементов  
- Работа с данными в структурах (массивы, объекты)  
- Обработка пользовательских событий (клики, ввод)  
- Простая модульная структура (ES modules)

# food-tracker
NutriHack — дневник питания, позволящий отслеживать съеденные калории и БЖУ. 
Приложение содержит дату, статус выполнения нормы по калориям, блоки приёмов пищи и кнопку добавления.
Пользователь может открывать модальное окно, вводить блюдо, получать БЖУ/ккал через API и сохранять результат в интерфейсе.

Макет Figma: https://www.figma.com/design/S9TPe5ATrjnax3sP7Wwxfx/UCD?node-id=114-662&m=dev
Деплой: https://elisabelousova.github.io/food-tracker/

Возможности
- Просмотр списка приёмов пищи (завтрак, обед, ужин, «ещё»)
- Модалка добавления блюда
- Автоподстановка БЖУ/ккал через **CalorieNinjas API**
- Автоматическая замена иконок на значения Б/Ж/У/ккал
- Подсчёт процента дневной нормы
